# 🌿 Naghea Hadulu Balecho Portfolio

This is a personal portfolio web app built with **HTML & CSS**, designed to showcase my professional background in **Procurement & Supply Chain Management**.  
The design uses earthy tones (olive green, terracotta, beige, and warm brown) to reflect grounded professionalism and reliability.

---

## ✨ Features
- **Landing Page (Home):** Hero section with name, role, tagline, and quick action buttons.
- **About Page:** Professional summary and values.
- **Experience Page:** Timeline of roles and achievements.
- **Contact Page:** Simple contact form and direct links.
- **Earthy Design Palette:** Warm, natural colors for a pleasing and professional look.

---
